# Create-Quiz-Backend




## 1. Description : Creates quiz with following parameters
```
date : Date
<!--- An array of mongoose subdocument -->
quiz : [{
  options:[{
    value:String,
    opted_by:Number
  }],
  answer:Number,
  answered_by:Number,
  question:String,
  hint:String
}]
```
## Method: POST
